# Proyecto EDA de sismos en python con jupyter notebook.
## Objetivo
    - Quiero generar un fichero .ipynb de jupyter notebook analizando los de una base de datos en mongodb.
    - El objetivo primero es extraer los datos de la bbdd el local de mongo Compass
    - El objetivo segundo es trasladar la información a un dataframe para poderlo manejar. 
    - El objetivo tercero seria limpiar la información del dataframe obtenido, de duplicados, nulos y valores incongruentes.
    - Objetivo final mostrar la información en graficas de los siguientes puntos:
          - Sismos más potentes ocurridos en el mundo segun su escala
          - Ciudades que más sismos sufren a lo largo del tiempo.
          - Mapa coropletico con las zonas mas sismicas del planeta.
    
# Resultado final
        - un fichero con extencion .ipynb comentado con markdown y conclusiones de los graficos pedidos.
